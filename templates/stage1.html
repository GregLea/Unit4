Stage 1

Tree like structure of HTML
HTML is structure a bit like a family tree. My parents had two kids, so there are two boxes underneath my parents box.
This forms the boxes of the webpage, which is the frame upon which all things on the web are hung.

Boxes, not Tubes
Everything on the web has a box that it fits into, and these boxes can be moved about on the page like pictures in a scrapbook.
There is an invisible grid that can be manipulated so make your HTML page look how you want it to look.

CSS
CSS stands for Cascading Style Sheets, which give coders a great way to control how their webpage looks.
By giving similar HTML elements the same class name, the CSS can tell the HTML what style it should use on that class.
